Looking back at our question, we see that it is asking for an elimination rate constant (Kel). We know that the units for Kel is usually per hour (sometimes written as hr -1 ). If we cross out units and don't end up with per hour, we've done something wrong.

This is going to be the elimination rate constant for {{drug.drug}}. We can look up the predicted value for the Kel of {{drug.drug}}. If our answer is not a number somewhat similar to the population average, we've probably done something wrong.

In this case, we are expecting a number that is probably between 0.001 and 0.300. If we end up with our answer being "57" or "0.00000034", we need to go back and check our calculations.
